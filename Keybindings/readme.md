# Keybindings Tweaks

**File to edit:** `~/.config/hypr/bindings.conf`

## Changes

### New App Bindings
- `SUPER + D` — Opens Nautilus directly to `~/Dropbox/Sem6` (quick access to semester folder)
- `SUPER + SHIFT + W` — Launches Typora (markdown editor)

### New Web App Bindings
- `SUPER + SHIFT + ALT + A` — Opens Grok (`https://grok.com`)
- `SUPER + SHIFT + ALT + G` — Opens WhatsApp Web (with focus-or-launch)
- `SUPER + SHIFT + CTRL + G` — Opens Google Messages (with focus-or-launch)
- `SUPER + SHIFT + P` — Opens Google Photos (with focus-or-launch)
- `SUPER + SHIFT + ALT + X` — Opens X compose/post page directly

### Monitor Toggle
- `SUPER + M` — Toggles the built-in laptop display using `~/.local/bin/toggle-edp.sh`

### Fullscreen Rebind
- Unbound default `SUPER + F` and `SUPER + ALT + F` to reassign them:
  - `SUPER + F` — Fullscreen (maximize, keeps gaps — mode `1`)
  - `SUPER + ALT + F` — True fullscreen (mode `0`)
